# LexicalAnalyzer
This is an implementation of a lexical analyzer in C++

## Requisites
[g++ 11](https://gcc.gnu.org/projects/cxx-status.html#cxx11)

## How to run
- one shall insert a regular grammar in ./expr/file.txt
- the program will look there to do its job
